# Transcriptomics Data Sources Guide

## Overview

There are multiple databases available for transcriptomics (RNA-seq) studies, which can be broadly categorized into **primary databases** and **supporting databases**.

In this guide, I focus on a few **core databases** that are most useful for beginners who want to start working with **bulk RNA-seq data**.

Understanding the structure of these databases, the type of data they provide, and how they are interconnected is essential before starting any analysis.

---

## Databases Covered

- GEO (Gene Expression Omnibus)
- SRA (Sequence Read Archive)
- ENA (European Nucleotide Archive)
